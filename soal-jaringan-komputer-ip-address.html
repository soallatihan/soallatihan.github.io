24. Apakah komputer yang terhubung ke jaringan boleh menggunakan alamat (ip address) yang sama?


Komputer mampu terhubung pada jaringan dan alamat yang sama adalah salah karena akan mengakibatkan konflik.

Pembasahan:

Protokol internet atau alamat IP adalah alamat pribadi Anda saat mengakses internet.  IP ini memungkinkan penyedia layanan Internet atau ISP Anda untuk menerima dan mengirim ulang aktivitas Internet yang dilakukan oleh seseorang. Selama Anda tidak berada di  jaringan yang sama, tidak ada masalah. Jika Anda berada di jaringan yang sama, konflik IP akan terjadi dan IP tersebut akan dianggap sebagai alamat yang buruk hingga salah satu atau keduanya mengubah IP. Tidak seperti alamat IP publik, alamat IP pribadi hanya berfungsi dalam jaringan area lokal (LAN). Menjadi lokal,  perangkat  dan tetangga  dapat menggunakan alamat IP pribadi yang sama. Artinya, alamat IP  adalah sekumpulan angka yang terkait dengan komputer, ponsel, atau perangkat "pintar" lainnya yang terhubung ke Internet. Nomor-nomor ini bervariasi dari perangkat ke perangkat dan digunakan untuk saling menghubungi.

38. Diketahui IP Address dari sebuah komputer server adalah 192.168.10.1/29 Jumlah sisa IP yang bisa digunakan oleh komputer lain yang satu jaringan dengan komputer server adalah​


Diketahui IP Address dari sebuah komputer server adalah 192.168.10.1/29 Jumlah sisa IP yang bisa digunakan oleh komputer lain yang satu jaringan dengan komputer server adalah​  5 buah IP address.

Penjelasan dengan langkah-langkah

Diketahui :

IP Address dari sebuah komputer server adalah 192.168.10.1/29

Ditanya : Jumlah sisa IP yang bisa digunakan oleh komputer lain yang satu jaringan dengan komputer server

Jawab :

∑ porsi jaringan = 29

∑ porsi host = 32 - 29 = 3

Bit terbelakang = 1₁₀ = 00000001₂

Bit host = 001

Bit host(network) = 000 (ubah semua 1 jadi 0)

Bit terbelakang(network) = 00000000₂ = 0₁₀

Bit host(broadcast) = 111 (ubah semua 0 jadi 1)

Bit terbelakang(broadcast) = 00000111₂ = 7₁₀

Network address = 192.168.10.0

Broadcast address = 192.168.10.7

Banyak IP dalam server = 7 - 0 = 7

2 ip address dipakai untuk network address dan broadcast address, sedangkan sisa nya digunakan oleh komputer lain

Jadi IP yang tersisa yang bisa digunakan komputer lain yaitu sebanyak 5 buah IP address
