25. x - 1 √ 1 - x⁴ Tolong gess soal mat minat


Fungsi f(x) = x - √(1 - x⁴) adalah fungsi yang valid untuk x yang berada dalam rentang (-1,1) karena jika x di luar rentang tersebut, maka nilai dari (1- x⁴) akan menjadi negatif dan akar dari bilangan negatif tidak dapat ditentukan.

1. Nilai Maksimum: Kita dapat menemukan nilai maksimum dengan mencari titik puncak dari grafik fungsi tersebut. Kita dapat menentukan titik puncak dengan mencari turunan pertama dari fungsi, yaitu f'(x) = 1 - (4x³) / √(1 - x⁴) . Kemudian, cari titik nol dari turunan pertama tersebut, yaitu x = 0, yang merupakan titik puncak dari grafik fungsi f(x) dan nilai maksimum adalah f(0) = 0

2. Nilai Minimum: Kita dapat menemukan nilai minimum dengan mencari titik dasar dari grafik fungsi tersebut. Kita dapat menentukan titik dasar dengan mencari turunan kedua dari fungsi, yaitu f''(x) = (12x²) / (1 - x⁴)^(3/2). Kemudian, cari titik nol dari turunan kedua tersebut, tidak ada titik nol pada turunan kedua, sehingga tidak ada nilai minimum pada fungsi f(x) = x - √(1 - x⁴)

3. Belok : Fungsi f(x) = x - √(1 - x⁴) tidak memiliki belok atau infleksi karena turunan kedua dari fungsi tersebut yaitu f''(x) = (12x²) / (1 - x⁴)^(3/2) tidak memiliki titik nol.

Jadi, fungsi f(x) = x - √(1 - x⁴) memiliki nilai maksimum pada x = 0 yaitu 0 dan tidak memiliki nilai minimum.

33. 3,5 + 0,5 x 1,5 - 0,3jawab dengan pakai carakalau berminat soal MTK saya nanti kasi soal lagi​


[tex]\bf3, 5 + 0, 5 \times 1, 5 - 0, 3 = \frac{35}{10} +( \frac{5}{10} \times \frac{15}{10} ) - \frac{3}{10} \\ \bf = \frac{35}{10} + ( \frac{1}{2} \times \frac{3}{2} ) - \frac{3}{10} = \frac{35}{10} + \frac{3}{4} - \frac{3}{10} \\ \bf = \frac{70 + 15 - 6}{20} = \frac{79}{20} = 3, 95[/tex]

Jawaban:

3,95

Pendahuluan:Pecahan Desimal

Pecahan Desimal adalah bilangan berkoma. Pecahan ini adalah bentuk yang paling sederhana dari pecahan biasa maupun pecahan campuran. Bilangan Berkoma tidak termasuk bilangan bulat. Karena memiliki sisa. Pecahan desimal didapat dari pembagian yang hasilnya bersisa/tidak bulat.

Pembahasan:

3,5 + 0,5 × 1,5 - 0,3

= 3,5 + (0,5 × 1,5) - 0,3

= 3,5 + 0,75 - 0,3

= 4,25 - 0,3

= 3,95
