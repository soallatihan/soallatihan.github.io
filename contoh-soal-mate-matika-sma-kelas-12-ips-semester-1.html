Penjelasan dengan langkah-langkah:

12/5 =

12/5 =

Mapel:Matematika

Kelas:6

materi:pecahan

Jawaban:

pendahuluan :

Pecahan memiliki beberapa macam, ada pecahan biasa, pecahan campuran,pecahan desimal, pecahan desimal.

pembahasan :

pecahan campuran :

[tex] \frac{12}{5} = 2 \frac{2}{5} [/tex]

pecahan desimal :

[tex] \frac{12 \times 2}{5 \times 2} = \frac{24}{10} = 2.4[/tex]

persen :

[tex] \frac{12 \times 20}{5 \times 20} = \frac{240}{100} = 240\%[/tex]

nama nama pecahan :

Pecahan memiliki beberapa macam, ada pecahan biasa, pecahan campuran,pecahan desimal, pecahan desimal.

1. pecahan biasa

pecahan biasa dilambangkan dengan tanda per (/), dalam bentuk huruf :

[tex]a/a = \frac{a}{a} [/tex]

2. pecahan campuran

pecahan campuran adalah gabungan dari bilangan bulat dan bilangan campuran.

dalam bentuk huruf :

[tex]a \frac{b}{c} [/tex]

3. pecahan desimal

pecahan biasa ditandai dengan tanda koma (,).

contoh :

[tex] \frac{2 \times 2}{5 \times 2} = \frac{4}{10} = 0.4[/tex]

4. persen

persen ditandai dengan lambang (%) ,persen artinya per seratus.

[tex] \frac{10}{100 } = 10\%[/tex]
