6. menentukan resultan gaya searah dan gaya berlawanan arah


Jika resultan gaya berlawanan maka rumusnya
resultan: F satu + F dua dan jika resultan gaya berlawanan maka rumusnya resultan : F satu-F dua

F: force (gaya) jika dua gaya segaris dan searah maka resultannya:
R = F1 + F2

jika dua gaya segaris dan berlawanan arah maka resultannya:
R = F2 - F1

Kelebihan :

1. Penggambaran vektor sesuai dengan skala sebenarnya.

3. Besar vektor sesuai dengan panjangnya.

Dapat menimbulkan kesalahan sistematis ketika menghitung dua vektor atau lebih.

Metode Analitik :

Kelebihan :

Nilai yang dihasilkan adalah exact (Tepat).

Kekurangan :

Harus hapal rumus dan memerlukan banyak waktu untuk menghitung rumusnya.

Pembahasan Soal :
Percepatan dan arah benda adalah...

Pembahasan Jawaban :
DiKetahui :
F1 = 40 N ( Kearah Kiri )
F2 = 40 N Cos 53° 
F3 = 84 N ( Kearah Kanan )
m = 6 Kg 

DiTanya :
a =...?

DiJawab : 
Sebelum mencari a. Kita harus mencari ∑F 
Rumus mencari ∑F
Yaitu :
∑F = F1 + F2 - F3 
∑F = 40 + ( 40.3/5 ) - 84
∑F = 40 + 24 - 84
∑F = 64 - 84
∑F = - 20 N ( Kearah Kiri )

Rumus mencari a
Yaitu :
∑F = m × a
- 20 = 6 × a 
a = - 20 / 6
a = - 3,3 m/s² ( Kearah Kiri )
Jadi, Percepatan benda dan arahnya adalah - 3,3 m/s² ( Kiri ) Opsi D.

23. contoh soal "Menentukan resultan kedua gaya pada sudut 90°" pakai cara


VEKTOR
• resultan

Sebuah benda dipengaruhi oleh 2 buah gaya F₁ dan F₂ saling tegak lurus.
Tentukan resultan yang dialami benda !

Tegak lurus → α = 90°
sehingga cos α = 0

Resultan kedua gaya
R² = F₁² + F₂² + 2 F₁ F₂ cos α
R² = F₁² + F₂²
R = √(F₁² + F₂²)


jika

a][ F₁ = 3 N dan F₂ = 4 N
→ R = √(3² + 4²) = 5 N

b][ F₁ = 5 N dan F₂ = 12 N
→ R = √(5² + 12²) = 13 N

c][ F₁ = 7 N dan F₂ = 24 N
→ R = √(7² + 24²) = 25 N

d][ F₁ = 8 N dan F₂ = 15 N
→ R = √(8² + 15²) = 17 N

e][ F₁ = 9 N dan F₂ = 40 N
→ R = √(9² + 49²) = 41 N
