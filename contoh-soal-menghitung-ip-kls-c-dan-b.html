13. Hitunglah subnetting IP Address Kelas C dari IP 192.10.10.10 a. Jumlah Subnet​


Untuk melakukan subnetting pada alamat IP Kelas C 192.10.10.10, kita akan membagi alamat IP ini menjadi beberapa subnet. Jumlah subnet yang dapat dibuat tergantung pada seberapa banyak bit host yang akan dialokasikan ke dalam subnet dan berapa banyak bit yang akan digunakan untuk subnetting.

Pada alamat IP Kelas C, biasanya, 24 bit pertama (dari bit 0 hingga bit 23) adalah bagian dari alamat jaringan, dan 8 bit terakhir (dari bit 24 hingga bit 31) adalah bagian dari alamat host. Namun, kita dapat membagi alamat IP ini lebih lanjut dengan meminjam beberapa bit dari alamat host untuk digunakan sebagai subnet.

Misalnya, jika kita ingin menggunakan 4 bit untuk subnetting, maka kita akan meminjam 4 bit dari alamat host, sehingga tersisa 4 bit untuk host. Ini akan menghasilkan 2^4 = 16 subnet yang berbeda.

Jadi, jumlah subnet yang dapat dibuat adalah 16.

17. tentukan ip address 192.168.10.100/30 a) ip network b)ip broadcast c) ip host d)sub netmask​


Jawaban:

Untuk alamat IP 192.168.10.100/30:

a) IP Network: 192.168.10.100/30 memiliki alamat IP jaringan 192.168.10.100.

Pada subnet /30, dua bit terakhir pada oktet keempat (100) digunakan untuk alamat host, sehingga alamat IP jaringannya adalah 192.168.10.100.

b) IP Broadcast: IP broadcast pada subnet /30 adalah alamat IP terakhir dalam jaringan. Dalam hal ini, alamat IP broadcast adalah 192.168.10.103.

c) IP Host: IP host dalam subnet /30 adalah alamat IP di antara IP jaringan dan IP broadcast. Dalam hal ini, alamat IP host yang valid adalah 192.168.10.101 dan 192.168.10.102.

d) Subnet Mask: Subnet mask untuk /30 adalah 255.255.255.252. Subnet mask ini digunakan untuk membagi alamat IP menjadi alamat jaringan dan alamat host.

21. 10 contoh ip lokal dan 10 contoh ip publik ?


Jawaban:

IP Address Public

IP public adalah alamat IP yang digunakan dalam jaringan global Internet serta penggunaan dan alokasinya diatur oleh InterNIC untuk menjamin penggunan IP address ini secara unik. Karena kelas IP address ini digunakan di dalam jaringan internet maka IP ini bisa diakses melalui jaringan internet secara langsung. Perangkat yang menggunakan IP public, seperti web server, mailserver, DNS server, game server ataupun perangkat lain dapat diakses dari jaringan manapun di dunia ini yang terkoneksi ke internet. Untuk dapat menggunakan IP public, suatu organisasi biasanya dapat mendaftarkan diri ke salah satu ISP (Internet Service Provider).

IP Address Private

Disebut IP address private karena IP ini hanya dikenali dan bisa diakses dari jaringan local saja dan tidak bisa diakses melalui jaringan internet secara langsung tanpa bantuan router yang mempunyai fitur NAT.

Penjelasan:

IP private digunakan untuk jaringan lokal agar sesama komputer dapat saling berkomunikasi, misalnya digunakan di jaringan sekolah, kantor, toko, warnet dan sebagainya. Perangkat yang terhubung ke jaringan lokal seperti printer, komputer, laptop, smartdevice menggunakan biasanya akan mendapatkan IP address private. Agar IP private dapat terhubung ke internet maka diperlukan router yang mempunyai kemampuan untuk melakukan NAT (Network Address Translation) agar semua device dengan IP private dapat terkoneksi ke internet dengan menggunakan IP public yang terkoneksi langsung ke Internet). Meskipun sudah terkoneksi ke internet, IP private tetap tidak bisa diakses langsung dari jaringan internet

28. Tuliskan dan beri contoh IP berdasarkan Kelas A, B, dan C!


Berikut adalah contoh IP berdasarkan kelasnya:

a. Kelas A IP:

Jangkauan nilai awal: 1.0.0.0 - 126.0.0.0

Contoh: 10.0.0.1, 45.12.123.65, 100.5.6.7

b. Kelas B IP:

Jangkauan nilai awal: 128.0.0.0 - 191.255.0.0

Contoh: 130.45.0.1, 172.16.0.2, 190.123.0.56

c. Kelas C IP:

Jangkauan nilai awal: 192.0.0.0 - 223.255.255.0

Contoh: 192.168.0.1, 200.50.100.5, 223.45.23.10

Pembahasan

IP (Internet Protocol) adalah protokol jaringan yang digunakan untuk mengirimkan dan menerima data melalui jaringan internet. IP bekerja pada layer internet dalam model referensi OSI (Open Systems Interconnection) dan bertanggung jawab untuk mengirimkan paket data antara alamat IP sumber dan tujuan. IP merupakan bagian penting dalam infrastruktur internet dan memungkinkan komunikasi antara perangkat yang berbeda di seluruh dunia.

36. diketahui ip address 192.168.10.100 tuliskan a. jumlah subnet b. ip validc. ip broadcast​


Untuk menjawab pertanyaan Anda, diperlukan informasi tambahan tentang subnet mask yang digunakan dalam jaringan tersebut. Subnet mask diperlukan untuk mengidentifikasi subnet, alamat IP yang valid, dan alamat IP broadcast.

Misalkan kita anggap subnet mask yang digunakan adalah 255.255.255.0 (CIDR: /24), berikut adalah jawaban untuk setiap pertanyaan:

a. Jumlah Subnet:

Jumlah subnet dapat ditentukan dengan melihat jumlah bit yang digunakan untuk subnet pada subnet mask. Dalam kasus ini, subnet mask adalah 255.255.255.0, yang berarti ada 24 bit yang digunakan untuk subnet dan 8 bit yang digunakan untuk host. Dengan 24 bit subnet, jumlah subnet yang mungkin adalah 2^24 - 2 (2^24 mengurangi subnet yang terreservasi untuk alamat jaringan dan alamat broadcast). Jadi, jumlah subnet yang mungkin adalah 2^24 - 2 = 16.777.214 subnet.

b. Alamat IP Valid:

Alamat IP yang valid dalam subnet ini adalah alamat yang digunakan untuk host. Dalam kasus ini, alamat IP 192.168.10.100 adalah alamat host, jadi itu adalah alamat IP yang valid dalam subnet tersebut.

c. Alamat IP Broadcast:

Alamat IP broadcast adalah alamat yang digunakan untuk mengirim pesan ke semua host dalam subnet. Dalam kasus ini, alamat IP broadcast dapat ditentukan dengan mengatur semua bit host pada alamat IP menjadi 1. Jadi, alamat IP broadcast untuk subnet tersebut adalah 192.168.10.255.
