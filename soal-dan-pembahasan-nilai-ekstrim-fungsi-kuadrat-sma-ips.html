23. Fungsi kuadrat f(x)=x²-2x-8 tentukan Titik potong dengan sumbu x, tentukan Titik potong dengan sumbu y, nilai ekstrim dan titik ekstrim, gambar grafik fungsi kuadrat​


Diketahui fungsi kuadrat :

f(x) = x² - 2x - 8

Titik potong sumbu x jika f(x) = 0, maka :

f(x) = x² - 2x - 8

0 = x² - 2x - 8

x² - 2x - 8 = 0

( x - 4 )( x + 2 ) = 0

x1 = 4 , x2 = -2

Titik potong sumbu y jika x = 0, maka :

f(x) = x² - 2x - 8

y = x² - 2x - 8

y = 0² - 2(0) - 8

y = 0 - 0 - 8

y = -8

Sumbu simetrinya :

sumbu simetri = ( x1 + x2 ) / 2

sumbu simetri = ( 4 + (-2) ) / 2

sumbu simetri = 2/2

sumbu simetri = 1

Titik puncak x :

x puncak = sumbu simetri

x puncak = 1

x puncak = 1

Titik puncak y :

f(x) = x² - 2x - 8

f(x puncak) = (x puncak)² - 2(x puncak) - 8

f(1) = (1)² - 2(1) - 8

f(1) = 1 - 2 - 8

f(1) = -9

y puncak = -9

Titik ekstrim fungsi :

titik ekstrim = titik puncak

titik ekstrim = ( x puncak , y puncak )

titik ekstrim = ( 1 , -9 )

24. nilai ekstrim dari fungsi kuadrat f(x)=-8x² -16x-1 adalah​


Jawaban:

Untuk menemukan nilai ekstrim dari fungsi kuadrat f(x)=-8x² -16x-1, kita dapat menggunakan rumus untuk menentukan vertex (titik puncak) dari parabola yaitu x = -b/2a dan substitusi nilai x tersebut ke dalam f(x).

Dalam hal ini, a = -8 dan b = -16, sehingga x = -(-16)/(2(-8)) = 1. Substitusi ke dalam f(x) menghasilkan:

f(1) = -8(1)² - 16(1) - 1 = -25

Sehingga nilai ekstrim dari fungsi kuadrat f(x) adalah -25, yang dicapai pada titik puncak parabola.

39. diketahui fungsi kuadrat yang grafiknya melalui titik A(0,4) B(-1,7) C(1,3) nilai ekstrim funsi kuadrat itu adalah


Dikarenakan melalui tiga titik sembarang, maka fungsinya dapat dinyatakan dalam:
f(x) = ax² + bx + c

Apabila disubstitusikan masing-masing dengan titik tersebut, akan terbentuk SPL tiga variabel.

Dengan melalui A(0,4):
4 = a.0² + b.0 + c
c = 4 ... (1)

Melalui B(-1,7)
7 = a.(-1)² + b.(-1) + c
7 = a - b + 4
3 = a - b
b = a - 3 ... (2)

Melalui C(1,3)
3 = a.1² + b.1 + c
3 = a + b + 4
-1 = a + b
Substitusi (2) ke persamaan berikut, menghasilkan:
-1 = a + (a-3)
-1 = 2a - 3
2a = 2
  a = 1

Diperoleh a = 1, sehingga b = a - 3 = -2, dan c = 4

Didapat: f(x) = x² - 2x + 4

Nilai ekstrim sebuah fungsi kuadrat diberikan dalam:
= -D/4a
= - (b² - 4ac) / 4a
= - ((-2)² - 4(1)(4)) / 4(1)
= - (4 - 16)/4
= 12/4
= 3

Sehingga, nilai ekstrin fungsi kuadrat tersebut adalah 3.
