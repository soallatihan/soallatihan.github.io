28. Contoh soal rotasi dalam geografi

a. x - 5y – 4 = 0

b. x + 5y + 4 = 0

c. 5x + 5y – 4 = 0

d. 5x - 5y – 4 = 0

e. x + 5y – 4 = 0

*Pembahasan* :

(x, y) o (y, -x)

x’ = y , y’ = -x

x’ = 5(-y’) + 4

x’ + 5y’ – 4 = 0 Jadi bayangan x + 5y – 4 = 0
