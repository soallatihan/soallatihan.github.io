19. banyaknya suku pada deret geometri 3+12+...+un=4095


Jawabannya ada di foto

Banyaknya suku pada deret geometri [tex] \rm 3+12+...+U_n = 4.095 [/tex] adalah 6 suku

Penjelasan dengan langkah-langkah

Barisan geometri adalah barisan bilangan yang memiliki rasio yang sama tiap suku selanjutnya. Deret geometri adalah penjumlahan urutan bilangan yang memiliki rasio yang sama.

Deret adalah jumlah keseluruhan bilangan yang memiliki aturan tertentu.

[tex] \rm U_1+U_2 + U_3+U_4+U_5+....+U_n [/tex]

Rumus rumus Barisan Geometri

[tex] \rm U_{n } = a \times r^{n-1} [/tex]

[tex] \rm r = \dfrac{ U_{2}}{U_{1} } ~~atau~~r = \dfrac{ U_{n}}{U_{n-1} } [/tex]

[tex] \rm a = U_{1} [/tex]

Rumus Deret Geometri

Untuk r > 1

[tex] \rm S_{n} = \dfrac{ a(r^{n} -1)}{ r-1} [/tex]

Untuk r < 1

[tex]\rm S_{n} = \dfrac{ a(1 - r^{n} )}{ 1 -r} [/tex]

Diketahui :

Suku pertama = a = 3Rasio = [tex] \rm \dfrac{ 12 }{3 } = 4 [/tex]Jumlah deret = Sn = 4.095

Ditanyakan :

Banyaknya suku deret geometri tersebut

Penyelesaian :

Langkah 1

Gunakan rumus deret geometri.

[tex] \rm S_n = \dfrac{ a(r^n-1) }{ r-1 } [/tex]

Subtitusi data yang diketahui.

[tex] \rm 4.095 = \dfrac{ 3(4^n-1) }{ 4-1 } [/tex]

Langkah 2

Tukar ruas kanan menjadi di sebelah kiri.

[tex] \rm \dfrac{ 3(4^n-1) }{ 4-1 } =4.095 [/tex]

Evaluasi penyebutnya.

[tex] \rm \dfrac{ 3(4^n-1) }{ 3 } =4.095 [/tex]

Karena pembilang dan penyebutnya sama sama memiliki 3, maka dapat kita bagi habis.

[tex] \rm \dfrac{ 1(4^n-1) }{ 1 } =4.095 [/tex]

[tex] \rm 4^n-1 =4.095 [/tex]

Langkah 3

Jumlahkan kedua ruas dengan 1

[tex] \rm 4^n-1 +1=4.095 +1 [/tex]

[tex] \rm 4^n=4.095 +1 [/tex]

[tex] \rm 4^n=4.096 [/tex]

Langkah 4

Ubah 4.096 menjadi bilangan berpangkat dengan basis 4, sehingga akan sama seperti ruas kiri.

[tex] \rm 4^n=4^6 [/tex]

Karena basis sama, syarat agar persamaan tersebut bernilai benar, maka pangkatnya harus sama, sehingga :

[tex] \rm 4=4 ~dan ~n=6 [/tex]

Langkah 5

Kebalikan n = 6 pada rumus deret awal.

[tex] \rm 4.095 = \dfrac{ 3(4^n-1) }{ 4-1 } [/tex]

[tex] \rm 4.095 = \dfrac{ 3(4^6-1) }{ 4-1 } [/tex]

[tex] \rm S_6 = \dfrac{ 3(4^6-1) }{ 4-1 } [/tex]

[tex] \rm S_6 = 4.095 [/tex]

[tex] \rm \bold{ \underbrace{U_1+U_2+U_3+U_4+U_5+U_6}_{ada ~6~ suku}} = 4.095 [/tex]

Jawaban Akhir dan Kesimpulan :

Jadi, banyaknya suku pada deret geometri tersebut adalah 6 suku

25. Suatu deret geometri diketahui suku ke-n dirumuskan dengan Un=23-2n. Rasio deret tersebut adalah


Untuk mencari rasio deret geometri, kita perlu menggunakan rumus rasio:

r = U<sub>n+1</sub> / U<sub>n</sub>

Dalam deret geometri ini, suku ke-n adalah:

U<sub>n</sub> = 23 - 2n

Suku berikutnya, yaitu suku ke-(n+1), adalah:

U<sub>n+1</sub> = 23 - 2(n+1) = 23 - 2n - 2

Substitusi kedua rumus tersebut ke rumus rasio akan menghasilkan:

r = (23 - 2(n+1)) / (23 - 2n)

r = (23 - 2n - 2) / (23 - 2n)

r = (21 - 2n) / (23 - 2n)

Jadi, rasio deret geometri tersebut adalah (21 - 2n) / (23 - 2n).
