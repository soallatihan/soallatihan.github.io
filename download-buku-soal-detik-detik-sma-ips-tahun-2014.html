4. Tahun 1940 satu komputer mengerjakan sekitar 100 operasi perdetik. Sejak itu kecepatan berlipat ganda 10 kali setiap 5 tahun. Berapa operasi komputer pada tahun 2014


Jawaban:

10¹⁵

Penjelasan dengan langkah-langkah:

Cari n

2014 - 1940 = 74

Karena selalu berkelipatan setiap 5 tahun sekali maka didapat

n = 74 / 5

n = 14 sisa 4 tahun (sisa tahun dapat diabaikan karena tidak termasuk kelipatan)

nilai ke n deret geometri

[tex] = a{r}^{n - 1} \\ = 100 \times {10}^{14 - 1} \\ = {10}^{2} \times {10}^{13} \\ = {10}^{15} [/tex]

Jadi, kecepatan operasi komputer tahun 2014 adalah 10¹⁵ operasi /detik

21. pak danang mendownload sebuah vidio yang berukuran 50mb dengan kecepatan download internet 100kbps, berapa detikkah waktu yang dibutuhkan pak danang untuk mendownload vidio tersebut?


Bila dikonversikan: 1 mega byte = 1.024 kilo byte
maka 50 mega byte = 51.200 kilo byte
kecepatan download = 100 kilo byte per second
maka waktu yang dibutuhkan: ukuran file / kecepatan download = 51.200 / 100 =512 second.

Jadi waktu yang dibutuhkan pak danang adalah 512 detik (kira-kira 8 menit)

32. Banyak buku di perpustakaan Kota Malang dari tahun 2014 sampai 2017 disajikanpada diagram batang di bawah ini. Berapa jumlah buku dari tahun 2014 hingga 2017Berapa selisih buku di tahun 2014 dan 2017?​


Dari diagram batang pada soal, maka diperoleh:

Jumlah buku dari tahun 2014 hingga 2017 adalah 1.800 buku.Selisih buku di tahun 2014 dan 2017 adalah 300 buku.

Penyajian data adalah suatu kegiatan penyampaian data yang diperoleh dalam bentuk diagram, tabel, maupun simbol. Penyajian data pada soal adalah penyajian data dalam bentuk diagram batang. Diagram batang adalah penyajian data dalam bentuk persegi panjang yang menjulang sesuai frekuensinya.

Penjelasan dengan langkah-langkah:

Diketahui:

Data banyak buku di perpustakaan kota Malang

Tahun 2014 = 300 buku

Tahun 2015 = 400 buku

Tahun 2016 = 500 buku

Tahun 2017 = 600 buku

Ditanya:

Jumlah buku dari tahun 2014 sampai 2017Selisih buku di tahun 2014 dan 2017

Jawab:

>> Hitung jumlah seluruh buku dari tahun 2014 sampai 2017

Jumlah buku

= 300 + 400 + 500 + 600

= 1.800 buku

>> Hitung selisih buku di tahun 2014 dan 2017

Selisih

= Banyak buku di tahun 2017 - Banyak buku di tahun 2014

= 600 - 300

= 300 buku

Kesimpulan:

Jadi, jumlah seluruh buku dari tahun 2014 sampai 2017 adalah 1.800 buku, dan selisih buku di tahun 2014 dan 2017 adalah 300 buku.
