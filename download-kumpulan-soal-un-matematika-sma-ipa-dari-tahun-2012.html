5. Nilai dari ∫ (3 sin 2x – cos x) dx, dengan x = 0 sampai x = (1/2)n adalah … (Ujian Nasional Matematika Tahun 2012) **Super Intensif UN (Ujian Nasional) SMA/MA IPA , Penerbit: Bisa!Publishing 2016, Penulis: Muhammad Amien & Team


Integral Trigonometri

∫ (3 sin 2x - cos x) dx, untuk batasan x = 0 → x = (1/2)π

⇔ (-3/2 cos 2x - sin x) → x = 0 dan x = (1/2)π

⇔ (3/2 - 1) - (-3/2 - 0)

⇔ 2

Jadi, integral dari persamaan trigonometri tersebut dengan batas x = 0 → x = (1/2)π adalah 2.

29. Diketahui titik A(1, 0, -2), B(2, 1, -1), C(2, 0, -3). Sudut antara vector AB dengan AC adalah … (Ujian Nasional Matematika Tahun 2012) **Super Intensif UN (Ujian Nasional) SMA/MA IPA , Penerbit: Bisa!Publishing 2016, Penulis: Muhammad Amien & Team


Diketahui titik A(1, 0, -2), B(2, 1, -1), C(2, 0, -3). Sudut antara vector AB dengan AC adalah 90°.

Pembahasan

Hai, kali ini materi yg akan kita bahas adalah Perkalian Skalar Dua Vektor.

Hasil kali skalar vektor a dan b merupakan suatu skalar yang besarnya sama dengan jumlah dari hasil kali komponen-komponen a dan b yang bersesuaian, ditulis a • b, dimana titik tersebut menandakan perkalian yg dibaca dot.

------> Panjang vektor a :

|a| = √(a₁² + a₂² + a₃²)

------> Hasil kali skalar vektor a dan b :

a • b = (a₁, a₂, a₃) • (b₁, b₂, b₃)

a • b = a₁b₁ + a₂b₂ + a₃b₃

------> Jika terdapat sudut antar vektor a dan b :

a • b = |a| |b| cos θ, dgn 0 ≤ θ ≤ π

------> Mencari sudut antara vektor a dan b :

[tex] \mathsf{cos \: \theta \: = \: \frac{ \vec{a} \: \cdot \: \vec{b}}{ |\vec{a}| \: |\vec{b}|}}[/tex]

Nah, untuk lebih jelasnya, kita simak penyelesaian soal tersebut !!!

••••••••••••

Diketahui :

Koordinat titik A(1, 0, -2), B(2, 1, -1), C(2, 0, -3)

Ditanya :

Sudut antara vektor AB dengan AC ?

Jawab :

Pertama, kita cari nilai vektor AB dan AC. Lalu, langsung kita cari panjang masing-masing vektor.

AB = B - A

AB = (2, 1, -1) - (1, 0, -2)

AB = (1, 1, 1)

|AB| = √(1² + 1² + 1²)

|AB| = √3

---------------

AC = C - A

AC = (2, 0, -3) - (1, 0, -2)

AC = (1, 0, -1)

|AC| = √(1² + 0² + (-1)²)

|AC| = √2

Selanjutnya, substitusikan hasil vektor tersebut ke dalam rumus mencari besar sudut.

[tex]\mathsf{cos \: \theta \: = \: \frac{ \vec{AB} \: \cdot \: \vec{AC}}{ |\vec{AB}| \: |\vec{AC}|}}[/tex]

⇒ [tex]\mathsf{cos \: \theta \: = \: \frac{ (1, \: 1, \: 1) \cdot \: (1, \: 0, \: -1) }{\sqrt{3}\: \cdot \: \sqrt{2} }}[/tex]

⇒ [tex]\mathsf{cos \: \theta \: = \: \frac{1\: + \: 0 \: + \: ( - 1)}{ \sqrt{6}}}[/tex]

⇒ [tex]\mathsf{cos \: \theta \: = \: \frac{0}{\sqrt{6}}}[/tex]

⇒ cos θ = 0

⇒ θ = 90°

Cos yg memiliki hasil 0 adalah cos sudut 90°.

Jadi, besar sudut antara vektor AB dan AC adalah 90°.
