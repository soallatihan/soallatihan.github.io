3. apa arti simbol fisika ini XL fisika​


Arti simbol XL fisika adalah reaktansi induktif

Pembahasan :

Simbol tersebut digunakan pada rangkaian seri RLC. Beberapa hal yang perlu diperhatikan :

XL = XC → bersifat resistifXL<XC → bersifat kapasitifXL>XC → bersifat induktif

XC adalah reaktansi kapasitif. Untuk mencai besarnya XL, dapat menggunakan rumus :

[tex]\boxed {\rm XL = \omega L}[/tex]

Keterangan :

XL = reaktansi indukstif (ohm)

ω = frekuensi sudut

L = induktansi Henry (H)

20. Nilai x yang memenuhi persamaan l2-xl²-l2-xl=3l2-xl+5 adalah​


Penjelasan dengan langkah-langkah:

|2 - x|² - |2 - x| = 3|2 - x| + 5

misal |2 - x| = p, maka :

p² - p = 3p + 5

p² - p - 3p - 5 = 0

p² - 4p - 5 = 0

(p + 1)(p - 5) = 0

p = -1 atau p = 5

• untuk p = -1

|2 - x| = -1 (tidak memenuhi, karena nilai mutlak tidak mungkin bernilai negatif)

• untuk p = 5

|2 - x| = 5

2 - x = -5 atau 2 - x = 5

-x = -5 - 2 atau -x = 5 - 2

-x = -7 atau -x = 3

x = 7 atau x = -3

Jadi, HP = {-3, 7}
