1/18 (4^6 - 1)
= 227,5

No. 1
Misal ;
u = 3x + 1
du/dx = 3
dx = 1/3 du

∫(3x + 1)^5 dx [1...0]
= ∫u^5 . (1/3) du
= (1/3) ∫ u^5 du
= 1/3 . 1/6 u^6
= (1/18)(3x + 1)^6
= (1/18)(3.1 + 1)^6 - (1/8)(3.0 + 1)^6
= (1/18)(4^6 - 1)
= 4095/18
= 227,5

No. 2
Misal :
u = 2x² + 1
du/dx = 4x
x dx = 1/4 du

∫x√(2x² + 1) dx [2...0]
= ∫u^1/2 . 1/4 du
= (1/4)(1/(1/2 + 1)) u^(1/2 + 1)
= 1/4 . 2/3 u^3/2
= 1/6 u^3/2
= 1/6 (2x² + 1)^3/2
= 1/6 (2.2² + 1)^3/2 - 1/6 (2.0² + 1)^3/2
= 1/6 . 9^3/2 - 1/6 . 1
= 1/6 . 3^3 - 1/6
= 1/6 (3^3 - 1)
= 1/6 . 26
= 13/3
= 4,333

15. soal UN 2013/2014 no.13


maaf harus di poto :)
clik gambar biar liat penuh :)Diket dan dit:
*n NaOH = 100 mL x 0,008 M = 0.8 mmol
*n CH3COOH = 100 mL x 0,008 M = 0,8 mmol
*Ka = 10^-5
*Ksp Fe(OH)2 = 1,6 x 10^-15
*[Fe2+] = ? M

A. Mencari nilai [OH-]
--> Soal UN ini mengaitkan 2 bab (kyk SBMPTN, ya), yaitu hidrolisis garam dan hasil kali kelarutan (Ksp), sehingga:
     CH3COOH + NaOH --> CH3COONa + H2O
m        0.8           0,8
b        -0.8          -0.8              +0.8          +0.8
s           -              -                 0.8

[G] = 0.8 mmol / 200 mL = 4 x 10^-3

Bersisa garamnya sebesar 0,8 mmol, masukkan ke rumus hidrolisis:
[OH-] = √Kw x [G]/Ka
         = √10^-14 x 4 x 10^-3 / 10^-5
         = 2 x 10^-6

B. Mencari [Fe2+] dari Fe(OH)2
--> Untuk mencarinya, buat reaksi ionisasi Besi (II) hidroksida tsb:
Fe(OH)2 ⇄ Fe2+ + 2OH-

Ksp = [Fe2+] [OH-]^2
1,6 x 10^-15 = [Fe2+] [2 x 10^-6]^2
[Fe2+] = 1,6 x 10^-15 / 4 x 10^-12
           = 0.4 x 10^-3
           = 4 x 10^-4 Molar (D)
