21. soal fisika kls x mipa 1gerak harmonik...​


Jawaban:

1. E. 4,90

2. D. 5,0

3. B. (1), (3), (4)

4. A. 4 J

5. B. 1

Penjelasan:

1. Besar gaya pemulih = F = [tex]\frac{-mgx}{l}[/tex]

(tanda negatif hanya menunjukkan bahwa gaya pemulih berlawanan arah dengan arah gerak bandul).

Keterangan: x = simpangan (m)

                     l = panjang tali

                     m = massa (kg)

                     g = percepatan gravitasi (m/s²)

F = [tex]\frac{0,25(9,8)(0,04)}{0,02}[/tex] N

  = 0,25 · 9,8 · 2 N

  = 0,5 · 9,8 N

  = 4,90 N

2. F = -k · Δx (tanda negatif hanya menunjukkan arah gaya pemulih)

      = 100 · 0,05 N

      = 5,0 N

3. (1) Simpangan terbesar centilever adalah 5 cm -> BENAR

   (2) Periode centilever 0,4 s -> SALAH, periode centilever = 0,8 s

   (3) Frekuensi centilever 1,25 Hz -> BENAR, f = 1/T = 1/0,8 = 1,25 Hz

   (4) Saat t = 1,8 s, centilever melakukan 2,25 getaran -> BENAR

        f = n/t

  1,25 = n/1,8

       n = 1,25 · 1,8 = 2,25

Pernyataan yang tepat adalah pernyataan nomor (1), (3), dan (4).

4. F = mg

      = 4 · 10 N

      = 40 N

  EP = 1/2 · k · x²

       = 1/2 · (k · x) · x

       = 1/2 · F · x

       = 1/2 · 40 · 0,2 J

       = 4 J

5. Misalkan percepatan gravitasi di planet tersebut adalah g, dan percepatan gravitasi di Bumi adalah g'.

g = 1/4 · g'

[tex]T=2\pi\sqrt{\frac{l}{g}}\\ 2 = 2\pi\sqrt{\frac{l}{g} } \\1 = \pi\sqrt{\frac{l}{g} } \\1 = \frac{\pi^2l}{g} \\g = \pi^2l[/tex]             [tex]T'=2\pi\sqrt{\frac{l}{g'} } \\T' = 2\pi\sqrt{\frac{l}{4g} } \\T'^2 = 4\pi^2\frac{l}{4g} \\T'^2 = \frac{\pi^2l}{g} \\T'^2 = \frac{g}{g} = 1\\T' = 1 s[/tex]  

Periode jam bandul tersebut di bumi adalah 1 s.

24. Banyak siswa pada kelas XII MIPA 1, XII MIPA 2, dan XII MIPA 3 masing-masing adalah 30, 25, dan 20. Rata-rata hasil ulangan Matematika kelas XII MIPA 1 dan XII MIPA 2 adalah 72 dan 68. Jika rata-rata gabungan ketiga kelas tersebut adalah 72, maka rata-rata hasil ulangan Matematika pada kelas XII MIPA 3 adalah ....


Rumus mencari rata-rata gabungan :

[tex]x = \frac{x1 \times n1 + x2 \times n2 + x3 \times n3......}{n \: total} [/tex]

x = rata-rata gabungan

x1 = rata-rata 1

n1 = banyak data 1

x2 = rata-rata 2

n2 = banyak data 2

dan seterusnya...

n total = banyak data total

Maka, penyelesaian :

[tex]x = \frac{x1 \times n1 + x2 \times n2 + x3 \times n3}{n1 + n2 + n3} \\ 72 = \frac{72 \times 30 + 68 \times 25 + x3 \times 20}{30 + 25 + 20} \\ 72 = \frac{2160 + 1700 + 20x3}{75} \\ 72 \times 75 = 3860 + 20x3 \\ 5400 = 3860 + 20x3 \\ 20x3 = 5400 - 3860 \\ 20x3 = 1540 \\ x3 = \frac{1540}{20} \\ x3 = 77[/tex]

Jadi, rata-rata nilai hasil ulangan matematika pada kelas XII MIPA 3 adalah 77
