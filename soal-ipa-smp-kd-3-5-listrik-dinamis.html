4. SOAL IPA KELAS 9 SEMESTER 1BAB 5 LISTRIK DINAMIS DALAM KEHIDUPAN SEHARI-HARI HALAMAN 221 TOLONG DI ISI A, B, C, D​


Penyelesaian soal

a. Rangkaian Hambatan 1

[tex]\frac{1}{R_{p} } = \frac{1}{2} + \frac{1}{4} + \frac{1}{6} = \frac{6 + 2 + 3 }{12} = \frac{11}{12}[/tex]

[tex]Rp_{p} = \frac{12}{11}[/tex] = 1,09 Ω

b. Rangkaian Hambatan 2

Rs = 3Ω + 2Ω +4Ω = 9Ω

c. Rangkaian Hambatan 3

Rs = 2Ω + 4Ω = 6Ω

[tex]\frac{1}{R_{p} } = \frac{1}{6} + \frac{1}{6} = \frac{2}{6}[/tex]

[tex]R_{tptal} = \frac{6}{2}[/tex] = 3Ω

d. Rangkaian Hambatan 4

[tex]\frac{1}{R_{p} } = \frac{1}{3} + \frac{1}{6} = \frac{3}{6}[/tex]

[tex]R_{p} = \frac{6}{3}[/tex] = 2Ω

Rtotal = 2Ω + 4Ω = 6Ω

Pembahasan

Pada suatu rangkaian listrik, hambatan listrik juga dapat dipasang  secara seri dan paralel seperti pada lampu dan baterai. Pola pemasangan hambatan  listrik ini ternyata juga memengaruhi besar arus listrik yang mengalir  pada suatu rangkaian listrik.

Rangkaian Hambatan Listrik Seri

Pada rangkaian seri kuat arusnya bernilai sama tetapi tegangannya  berbeda-beda, sehingga:

V1  ≠ V2  ≠ ... ≠ Vn

Vtotal = V1  + V2  + ...+Vn

Menurut hukum Ohm

V1 = I ∙ R1

V2 = I ∙ R2

Vtotal = IR1+ IR2= I (R1  + R2)

Karena :

Vs  = I ∙ Rs   = Vtotal

maka, Rs

Rs  = R1  + R2

Jika ada n buah hambatan yang disusun secara seri maka:

Rs  = R1  + R2  + .... + Rn

Rangkaian Hambatan Listrik Paralel

Paralel  Pada rangkaian paralel,  tegangan listrik bernilai sama  tetapi besar kuat arusnya berbeda,  sehingga:

I1 ≠ I2 ≠ ... ≠ In

Menurut hukum Ohm

[tex]I_{1} = \frac{V}{R_{1} }[/tex] dan [tex]I_{2} = \frac{V}{R_{2} }[/tex]

[tex]I_{total} = I_{1} + I_{2}[/tex]

[tex]I_{total} = \frac{V}{R_{1} } + \frac{V}{R_2 } = V (\frac{1}{R_{1} } + \frac{1}{R_{2 }} )[/tex]

Karena

[tex]I_{total} = \frac{V}{R_{p} }[/tex]

maka,

[tex]\frac{1}{R_{p} } = \frac{1}{R_{1} } + \frac{1}{R_{2} }[/tex]

Sehingga, untuk n buah hambatan paralel:

[tex]\frac{1}{R_{p} } = \frac{1}{R_{1} } + \frac{1}{R_{2} } + ..... + \frac{1}{R_{n} }[/tex]

24. kerjakan soal tentang listrik dinamis​


Jawaban:

a. 11 ohm

b. 0,54 Ampere

Penjelasan:

a. 》Diketahui :

R1 = 2 ohmR2 = 3 ohmR3 = 6 ohm

Ditanya :

Besar hambatan penggantinya/ R Total ??

Penyelesaian :

RS = R1 + R2 + R3

= 2 + 3 + 6

= 5 + 6

= 11 ohm

Jadi, R Total = RS

- R Total = 11 ohm

Jadi, besar hambatan penggantinya/ R Total adalah 11 ohm.

b. 》Diketahui :

R = 11 Ohm

V = 6 Volt

Ditanya :

Kuat arus listrik ( I ) ...??

Penyelesaian :

I = V/ R

= 6 Volt / 11 Ohm

= 0,54 Ampere

Jadi, kuat arus listriknya adalah 0,54 Ampere.
