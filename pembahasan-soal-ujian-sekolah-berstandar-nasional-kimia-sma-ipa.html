La = Lb

Ia ωa = Ib ωb

{m ra²} ωa = {m rb²} ωb

Koordinat titik berat x

x = ( A1. x1 + A2. x2 )/ (A1 + A2)

x = ( 32. 2 + 16. 6 )/(32 + 16)

x = 160/48 = 10/3


Koordinat titik berat

y = ( A1. y1 + A2. y2 )/ (A1 + A2)

y = ( 32. 4 + 16. 2 )/(32 + 16)

y = 160/48 = 10/3


Koordinat titik berat di (10/3, 10/3)
