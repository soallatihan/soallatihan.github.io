8. Bentuk sederhana dari a3.b-2.c5:a-2.b7.c4 adalah


Kategori Soal : Matematika - Bentuk Pangkat
Kelas : X (1 SMA)
Pembahasan :
pᵃ x pᵇ = pᵃ ⁺ ᵇ
pᵃ : pᵇ = pᵃ ⁻ ᵇ
(pᵃ)ⁿ = pᵃ ˣ ⁿ

Mari kita lihat soal tersebut.
(a³ x b⁻² x c⁵) : (a⁻² x b⁷ x c⁴)
= (a³ : a⁻²) x (b⁻² : b⁷) x (c⁵ : c⁴)
= a³ ⁻ ⁽⁻²⁾ x b⁻² ⁻ ⁷ x c⁵ ⁻ ⁴
= a⁵ x b⁻⁹ x c¹
= a⁵ x c¹ : b⁹

Semangat!
