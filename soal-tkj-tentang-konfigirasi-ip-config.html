18. Apa yang dimaksud config dan kegunaannya


Pengertian config adalah kepanjangan dari configuration yang memiliki artian sebagai pengaturan. Configuration adalah sebuah fungsi yang telah dibuat pada sebuah program yang berada pada komputer yang berfungsi untuk dapat diatur sedemikian rupa oleh si pengguna untuk dapat menjadi sesuai dengan kebutuhan yang diharapkan.

Kegunaan yang berada pada config adalah untuk mengoptimalisasi tindakan yang dilakukan pada aplikasi. Umumnya, penggunaan config sendiri dilakukan pada server.

Pembahasan

Server adalah sebuah bentuk komputer yang telah dibuat untuk melakukan pemrosesan terhadap permintaan dan juga melakukan pengiriman data dari komputer lain dengan cara menggunakan internet maupun sebuah jaringan komputer. Fungsi dasar yang dimiliki oleh sebuah  server adalah untuk melakukan penerimaan terhadap permintaan yang berada pada jaringan yang masuknya. Contohnya yang paling akan dapat dengan mudah dimengerti adalah interaksi yang berada pada server web dan juga browser. Kemudian, server sendiri juga dianggap sebagai sebuah software yang melakukan pelaksanaan terhadap tugas tertentu. Akan tetapi hardware yang melakukan pendukungan terhadap software juga disebut sebagai server dikarenakan software server akan terhubung dengan ribuan client.

28. untuk melihat indikasi pada konfigurasi IP yang terpasang oada komputer dengan sistem operasi windows dengaa perintah?a. if configb. ip configc. is konfigd. IS config


Perintah yang digunakan untuk melihat indikasi pada konfigurasi IP yang terpasang pada komputer dengan sistem operasi Windows adalah IP Config.

Pembahasan

IP atau yang juga dikenal dengan sebutan Internet Protocol adalah serangkaian angka dan titik yang disematkan kepada setiap unit komputer dan perangkat jaringan. Konfigurasi angka dan titik ini berfungsi layaknya alamat pada jaringan komputer. Dengan kata lain, IP sebuah komputer atau perangkat jaringan bersifat unik atau terbatas untuk perangkat itu sendiri. Dengan demikian, ketika terhubung ke sebuah jaringan, sebuah komputer atau perangkat jaringan dapat dikenali secara khusus berkat IP yang tersemat kepadanya.

Pada perkembangannya, alamat atau identitas unik sebuah komputer atau perangkat jaringan tidak hanya terbatas pada IP melainkan juga pada MAC address yang berfungsi kurang lebih sama, namun MAC address turut mengombinasikan huruf selain angka dan titik.

40. lakukan konveksi bilangan berikutsoal jurusan TKJ ​


Jawab:

a. 110₍₁₀₎ = 6E₍₁₆₎b. 1111110₍₂₎ = 7E₍₁₆₎c. 89340₍₁₀₎ = 256374₍₈₎d. 653₍₈₎ = 110101011₍₂₎e. 110100101010101₍₂₎ = 64525₍₈₎f. ABC₍₁₆₎ = 5274₍₈₎

Penjelasan dengan langkah-langkah:

a. Aturan konversi bilangan berbasis 10 (desimal) ke bilangan berbasis 16 (heksadesimal):

- bagi bilangan desimal dg angka 16.

- bila langsung diketahui, hasilnya akhirnya dikonversi sesuai tabel sistem bilangan.

Caranya bisa dilihat di lampiran (sebelah tabel sistem bilangan)

b. Aturan Konversi bilangan berbasis 2 (biner)  ke bilangan berbasis 16 (heksadesimal):

- kelompokkan bilangan biner menjadi 4 digit (dari kanan)

- ubah bilangannya sesuai tabel sistem bilangan.

Sehingga caranya: 1111110 dikelompokkan menjadi sebagai berikut:

0111₍₂₎ = 7₍₁₆₎

1110₍₂₎ = E₍₁₆₎

Jadi 1111110₍₂₎ = 7E₍₁₆₎

c. Aturan konversi bilangan berbasis 10 (desimal) ke bilangan berbasis 8 (oktal):

- bagi bilangan desimal dg angka 8.

- tulis hasilnya dan hasil sisa baginya

- bila hasil akhirnya 7 atau kurang dari 7, hentikan pembagian

- penulisan konversi dari bawah (hasil akhir) ke atas (sisa pembagian)

Caranya bisa dilihat di lampiran (sebelah tabel sistem bilangan)

d. Konversi oktal ke biner dilakukan dg langsung mengubahnya sesuai tabel sistem bilangan, namun 1 digit oktal dituliskan 3 digit biner. Caranya

6₍₈₎ = 110₍₂₎

5₍₈₎ = 101₍₂₎

3₍₈₎ = 011₍₂₎

Jadi 653₍₈₎ = 110101011₍₂₎

e. Konversi biner ke oktal dilakukan dg mengelompokkan biner menjadi 3 digit (dari kanan), lalu mengubahnya ke oktal. Caranya:

110₍₂₎ = 6₍₈₎

100₍₂₎ = 4₍₈₎

101₍₂₎ = 5₍₈₎

010₍₂₎ = 2₍₈₎

101₍₂₎ = 5₍₈₎

Jadi 110100101010101₍₂₎ = 64525₍₈₎

f. Konversi heksadesimal ke oktal dilakukan dg aturan:

- tiap digit bilangan heksadesimal diubah ke biner

- 1 digit bilangan heksadesimal dituliskan 4 digit bilangan biner

- kelompokkan hasil konversi biner tadi menjadi 3 digit (dari kanan)

- ubah biner ke oktal sesuai tabel sistem bilangan. Caranya:

A₍₁₆₎ = 1010₍₂₎

B₍₁₆₎ = 1011₍₂₎

C₍₁₆₎ = 1100₍₂₎

Maka ABC₍₁₆₎ = 101010111100₍₂₎ dikelompokkan menjadi :

101₍₂₎ = 5₍₈₎

010₍₂₎ = 2₍₈₎

111₍₂₎ = 7₍₈₎

100₍₂₎ = 4₍₈₎

Jadi ABC₍₁₆₎ = 5274₍₈₎

Amatilah tabel sistem bilangan di lampiran.
