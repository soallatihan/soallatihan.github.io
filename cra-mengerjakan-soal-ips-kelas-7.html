6. Nilai f(x)+g(x) =ka tolong di bantu jwb dgn memberikan contoh soal cra mengerjakannya beserta jawabannya​


Jawaban:

f(x) + g(x) = ( f + g )(x)

Contoh soal

f(x) = 3x + 2

g(x) = 4x - 3

nilai (f+g)(x) ?

nilai (f+g)(2) ?

Jawab :

f(x) = 3x + 2

g(x) = 4x - 3

( f + g )(x)

= (3x + 2) + (4x - 3)

= 3x + 4x + 2 - 3

= 7x - 1

nilai (f+g)(2)

(f+g)(x) = 7x - 1

(f+g)(2) = 7(2) - 1

(f+g)(2) = 13

cara untuk menjumlahkan dekomposisi fungsi ini, dg cara menjumlahkan variabel sejenis
