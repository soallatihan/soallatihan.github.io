12. Perhatikan data berikut! Data nilai statistik 10 orang mahasiswa sebagai berikut: 50,40,70, 75, 75, 80,65, 75, 30, 80.Carilah nilai simpangan rata-ratanya! Berdasarkan data di atas, tentukanlah simpangan baku (standar deviasi) dan koefisien variasinya!​


Dari data 50,40,70, 75, 75, 80,65, 75, 30, dan 80 yang didapatkan dari 10 orang, berikut adalah hasil dari analisa terhadap data tersebut:

Simpangan rata-rata: -2.Simpangan baku: 19.26.Koefisien variasi: 29.63%.

Pembahasan

Berikut adalah cara dari mencari masing-masing nilai yang ada pada nilai statistika pada soal:

Simpangan rata-rata

Langkah pertama adalah mencari rata-rata atau mean dari nilai statistika tersebut, sehingga didapatkan:

(50 + 40 + 70 + 75 + 75 + 80 + 65 + 75 + 30 + 80) / 10 = 65

Setelah mendapatkan rata-tata dari data tersebut, maka kita bisa mencari simpangan rata-rata dengan cara: 

(50 - 65) + (40 - 65) + (70 - 65) + (75 - 65) + (75 - 65) + (80 - 65) + (65 - 65) + (75 - 65) + (30 - 65) + (80 - 65) / 10 = -20 /2 = -2.

Simpangan baku

Untuk mencari standar deviasi, cara yang digunakan adalah dengan cara mencari selisih kuadrat dari masing-masing data, seperti berikut:

√[((50 - 65)² + (40 - 65)² + (70 - 65)² + (75 - 65)² + (75 - 65)² + (80 - 65)² + (65 - 65)² + (75 - 65)² + (30 - 65)² + (80 - 65)²) / 10] = 19.26

Koefisien variasi

Cara yang digunakan untuk mencari koefisien variasi adalah dengan cara membagi nilai dari standar deviasi atau simpangan baku dengan rata-rata yang kemudian dikalikan dengan 100%, sehingga:

(19.26 / 65) x 100% = 29.63%.

30. Perhatikan data berikut! Data nilai statistik 10 orang mahasiswa sebagai berikut: 50,40,70, 75, 75, 80,65, 75, 30, 80. Carilah nilai simpangan rata-ratanya! Berdasarkan data di atas, tentukanlah simpangan baku (standar deviasi) dan koefisien variasinya!


Jawaban:

Langkah-langkah untuk mencari nilai simpangan rata-rata, simpangan baku (standar deviasi), dan koefisien variasi adalah sebagai berikut:

1. Hitung rata-rata data:

  Rata-rata = (50+40+70+75+75+80+65+75+30+80) / 10

                  = 60

2. Hitung simpangan setiap nilai data dengan rata-rata:

  Simpangan            = Nilai - Rata-rata

  Simpangan data 1 = 50 - 60 = -10

  Simpangan data 2 = 40 - 60 = -20

  Simpangan data 3 = 70 - 60 = 10

  Simpangan data 4 = 75 - 60 = 15

  Simpangan data 5 = 75 - 60 = 15

  Simpangan data 6 = 80 - 60 = 20

  Simpangan data 7 = 65 - 60 = 5

  Simpangan data 8 = 75 - 60 = 15

  Simpangan data 9 = 30 - 60 = -30

  Simpangan data 10 = 80 - 60 = 20

3. Hitung nilai simpangan rata-ratanya:

  Simpangan Rata-rata = ( -10 -20 +10 +15 +15 +20 +5 +15 -30 +20 ) / 10

                                      = 5

4. Hitung simpangan baku (standar deviasi):

  a. Hitung kuadrat dari setiap simpangan:

     Simpangan kuadrat data 1 = (-10)^2 = 100

     Simpangan kuadrat data 2 = (-20)^2 = 400

     Simpangan kuadrat data 3 = (10)^2 = 100

     Simpangan kuadrat data 4 = (15)^2 = 225

     Simpangan kuadrat data 5 = (15)^2 = 225

     Simpangan kuadrat data 6 = (20)^2 = 400

     Simpangan kuadrat data 7 = (5)^2 = 25

     Simpangan kuadrat data 8 = (15)^2 = 225

     Simpangan kuadrat data 9 = (-30)^2 = 900

     Simpangan kuadrat data 10 = (20)^2 = 400

 

  b. Hitung rata-rata dari simpangan kuadrat:

      Rata-rata simpangan kuadrat

      = (100+400+100+225+225+400+25+225+900+400) / 10

      = 330

  c. Hitung simpangan baku dengan rumus:

     Simpangan baku (standar deviasi) = akar dari Rata-rata simpangan kuadrat

                                                               = akar (330)

                                                               = 18,165

5. Hitung koefisien variasi:

  Koefisien variasi = (Simpangan baku / Rata-rata) x 100%

                               = (18,165 / 60) x 100%

                               = 30,275%

Jadi, nilai simpangan rata-rata adalah 5, simpangan baku (standar deviasi) adalah 18,165, dan koefisien variasi adalah 30,275%.
