17. Ukuran file yang akan didownload adalah 70MB (megabyte), sedangkan kecepatan internet adalah 3. 6Mbps (megabit per second). Maka untuk mengitung waktu yang butuhkan untuk mendownload file yang ukurannya adalah 70MB adalah.


Jawaban:

Untuk mengitung waktu yang dibutuhkan untuk mendownload file dengan ukuran 70MB, pertama-tama kita perlu mengkonversi kecepatan internet dari Mbps (megabit per second) ke MBps (megabyte per second). Hal ini dapat dilakukan dengan membagi kecepatan internet dalam Mbps dengan 8 (karena 1 byte adalah 8 bit). Dengan demikian, kecepatan internet dalam MBps adalah 3.6 / 8 = 0.45 MBps.

Setelah mengetahui kecepatan internet dalam MBps, kita dapat mengitung waktu yang dibutuhkan untuk mendownload file dengan ukuran 70MB dengan menggunakan rumus waktu = ukuran file / kecepatan internet. Dengan menggunakan rumus tersebut, kita dapat menghitung waktu yang dibutuhkan untuk mendownload file sebagai berikut:

waktu = 70MB / 0.45 MBps

waktu = 156.666667 detik

Jadi, waktu yang dibutuhkan untuk mendownload file dengan ukuran 70MB adalah 156.666667 detik atau sekitar 2,6 menit.

27. jika kita mempunyai kecepatan jaringan 7,2 Mbps dan kita ingin mendownload sebuah game dugaan ukuran penyimpanan 1,5 Gb berapa waktu yang dibutuhkan dalam mendownload game tersebut​


Jawaban:

Untuk menghitung waktu yang dibutuhkan dalam mendownload game, Anda perlu mengubah ukuran game dari gigabyte (GB) menjadi megabit (Mbps) untuk mencocokkan dengan kecepatan jaringan Anda. Kemudian, Anda dapat menggunakan rumus sederhana:

Waktu (detik) = (Ukuran dalam Megabit) / (Kecepatan dalam Mbps)

Langkah-langkahnya adalah sebagai berikut:

1. Ubah ukuran game dari GB ke MB:

1.5 GB = 1.5 * 1024 MB = 1536 MB

2. Hitung waktu yang dibutuhkan:

Waktu (detik) = 1536 MB / 7.2 Mbps ≈ 213.33 detik

Namun, kita lebih sering mengukur waktu dalam menit atau jam daripada detik dalam kasus seperti ini. Mari kita konversi hasil di atas ke dalam menit:

1 menit = 60 detik

Waktu (menit) = 213.33 detik / 60 ≈ 3.56 menit

Jadi, diperlukan sekitar 3.56 menit atau sekitar 3 menit 34 detik untuk mendownload game dengan ukuran penyimpanan 1.5 GB pada kecepatan jaringan 7.2 Mbps.
