40. Jawaban terbaik Kelas XIMatematika Wajib


110 = a + 3.8
a = 86

U30 = 86 + 29. 8 = 318Diketahui:
b = 8
U4 = 110

Ditanya:
U30 = ?

Jawaban:
Un - (n - 1)b = a
U4 - (4 - 1)b = a
U4 - 3b = a
110 - 3(8) = a
110 - 24 = a
a = 86

maka:
Un = a + (n - 1)b
U30 = 86 + (30 - 1)8
U30 = 86 + (29)8
U30 = 86 + 232
U30 = 318
