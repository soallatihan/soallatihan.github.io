10. fungsi tabungan (s) adalah... soal integral kelas 11​


Jawab:

Penjelasan dengan langkah-langkah:

Fungsi tabungan yaitu fungsi yang menunjukkan hubungan antara tabungan (S) dengan pendapatan (Y). Dengan menggunakan rumus fungsi konsumsi, dapat ditentukan sebagai berikut.

Y = C + S

S = Y – C padahal C = a + bY,

sehingga

S = Y – (a + bY)

S = Y – a – bY

S = -a + (1 – b) Y

Jadi, fungsi tabungan dapat dirumuskan sebagai berikut.

S = -a + (1 – b) Y

sedangkan

Syarat mutlak fungsi tabungan yaitu:

– nilai a = harus negatif

– nilai 1 – b = harus positif

Keterangan:

S = tingkat tabungan nasional

1 – b = MPS yaitu tambahan pendapatan yang digunakan untuk tambahan tabungan

★ persamaan kurva

• integral pertama → turunan pertama, y' = f'(x)

d²y/dx² = 18x - 8

y' = ʃ 18x - 8 dx

y' = 9x² - 8x + c1

• integral kedua → fungsi awal, y = f(x)

dy/dx (x) = 9x² - 8x + c1

y = ʃ 9x² - 8x + c1 dx

y = 3x³ - 4x² + c1x + c2

• mencari nilai konstanta, c dari kedua titik yang dilalui

a. titik A (0, 3)

y = 3x³ - 4x² + c1x + c2

3 = 3(0)³ - 4(0)² + c1(0) + c2

c2 = 3

b. titik B (1, -4)

y = 3x³ - 4x² + c1x + c2

-4 = 3(1)³ - 4(1)² + c1(1) + 3

-4 = 3 - 4 + c1 + 3

0 = 6 + c1

c1 = -6

sehingga, persamaan kurva di atas adalah:

y = 3x³ - 4x² + c1x + c2

→ substitusi nilai c1 dan c2, maka:

f(x) = y = 3x³ - 4x² - 6x + 3 (A)
