x+y=42
x=y+10

x+y=42

42-16= x

Untuk Toko Rame:
- Baju : 80.000 × 25/100 = 20.000. Jadi, 80.000 - 20.000 = 60.000
- Celana : 100.000 × 10/100 = 10.000. Jadi, 100.000 - 10.000 = 90.000
Jadi, harga total 60.000 + 90.000 = 150.000.

Untuk Toko Damai :
- Baju : 80.000 × 20/100 = 16.000. Jadi, 80.000 - 16.000 = 64.000
- Celana : 100.000 × 15/100 = 15.000. Jadi, 100.000 - 15.000 = 85.000.
Jadi, harga total = 64.000 + 85.000 = 149.000.

Untuk Toko Senang :
- Baju : 80.000 × 15/100 = 12.000. Jadi, 80.000 - 12.000 = 68.000.
- Celana : 100.000 × 20/100 = 20.000 .Jadi, 100.000 - 20.000 = 80.000.
Jadi, harga total = 68.000 + 80.000 = 148.000.
Untuk Toko Indah :
- Baju : 80.000 × 10/100 = 8.000. Jadi, 80.000 - 8.000 = 72.000.
- Celana : 100.000 × 25/100 = 25.000. Jadi, 100.000 - 25.000 = 75.000
Jadi, harga total = 72.000 + 75.000 = 147.000.

Kita dapat menyimpulakn bahwa toko yang menawarkan harga paling murah adalah = D. Tokoh Indah dengan harga total Rp. 147.000.
