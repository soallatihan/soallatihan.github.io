9. 2. ip address 100.100.100.100tentukan class :oktet network:oktet host:subner mask:ip network:ip host:ip bc :​


Alamat IP 100.100.100.100 adalah contoh alamat IP, dan kita dapat menganalisisnya sebagai berikut:

Class: Alamat IP ini adalah bagian dari kelas A, karena rentang alamat IP kelas A adalah 0.0.0.0 hingga 127.255.255.255.

Oktet Network: Pada alamat IP kelas A, oktet pertama (100) adalah bagian dari alamat jaringan.

Oktet Host: Tiga oktet berikutnya (100.100.100) adalah bagian dari alamat host.

Subnet Mask: Karena ini adalah alamat kelas A, subnet mask standarnya adalah 255.0.0.0.

IP Network: Alamat jaringan adalah 100.0.0.0 (dengan tiga oktet pertama dari alamat IP).

IP Host: Alamat host adalah 0.100.100.100 (dengan tiga oktet terakhir dari alamat IP).

IP Broadcast: Alamat broadcast dalam jaringan ini adalah 100.255.255.255 (dengan tiga oktet pertama tetap dan oktet terakhir diatur menjadi 255).

15. Tentukan network ID pada IP 192.168.20.1/24


Jawaban:

Dalam notasi IP 192.168.20.1/24, angka 24 pada akhir blok menunjukkan jumlah bit 1 yang ada pada network portion, atau dapat dikatakan jumlah bit yang digunakan untuk mengidentifikasi network ID. Dalam hal ini, karena menggunakan 24 bit untuk network ID maka tinggal 8 bit lagi untuk host ID.

Untuk menentukan network ID dari IP tersebut, perlu dilakukan operasi AND antara setiap oktet IP dengan subnet mask. Subnet mask 24 bit adalah 255.255.255.0, yang dapat ditulis dalam bentuk binner sebagai 11111111.11111111.11111111.00000000.

Maka jika dilakukan operasi AND dengan IP 192.168.20.1:

```

192.168.20.1 -> 11000000.10101000.00010100.00000001 (dalam biner)

& subnet mask (24) -> 11111111.11111111.11111111.00000000 (dalam biner)

-----------------------------------------------------------------

network ID -> 11000000.10101000.00010100.00000000 (dalam biner)

```

Biner 11000000.10101000.00010100.00000000 dapat dikonversi ke decimal untuk mendapatkan network ID-nya. Sehingga, network ID dari IP yang diberikan adalah 192.168.20.0.

25. ip address 172.10.100.200/25 berapa ip networknya


Jawaban:

Untuk mengetahui IP Network dari alamat IP 172.10.100.200/25, Anda perlu melakukan operasi bitwise AND antara alamat IP dan subnet mask yang sesuai. Subnet mask /25 akan memiliki nilai biner 1 pada 25 bit pertama dan 0 pada bit-bit yang tersisa.

Dalam hal ini, subnet mask adalah 255.255.255.128, atau dalam bentuk biner 11111111.11111111.11111111.10000000.

Maka operasi bitwise AND pada alamat IP 172.10.100.200 dengan subnet mask tersebut adalah sebagai berikut:

172.10.100.200       -> 10101100.00001010.01100100.11001000

255.255.255.128  -> 11111111.11111111.11111111.10000000

-----------------------------------------------------------

Network Address      -> 10101100.00001010.01100100.10000000

Jadi, IP Network dari alamat IP 172.10.100.200/25 adalah 172.10.100.128.

27. Network id, broadcast id, range ip, jumlah host dari ip 192.168.45.1/24


IP address : 192.168.45.1/24 atau

= 192.168.45.1 255.255.255.0

Cara menghitung :

Ubah IP Address dan Netmask kedalam bentuk biner, 192.168.45.1 IP Address tersebut berada di kelas C,  ambil octet ketiga dan ke empat untuk dihitung, maka :

IP address : x.x.0010 1101. 0000 0001 = 45.1

Netmask : x.x.1111 1111. 0000 0000 = 255.0

Jawab:

1. Network ID

Network ID = IP Address AND Netmask

IP address : x.x.0010 1101. 0000 0001

Netmask    : x.x.1111 1111. 0000 0000

——————————————————

AND           : x.x.0010 1101. 0000 0000 = 45.0

Sehingga Network ID nya adalah 192.168.45.0

2. Broadcast ID

Broadcast ID = IP Address OR ~Netmask

IP address : x.x.0010 1101. 0000 0001

~Netmask  : x.x.0000 0000 . 1111 1111

———————————————————

OR : x.x.0010 1101. 1111 1111 = 45.255

Sehingga Broadcast Id nya adalah 192.168.45.255

3. Range IP

Range IP = Network ID hingga(-) Broadcast ID

Sehingga Range IP nya adalah 192.168.45.0 – 192.168.45.255

4. Jumlah Host

Jumlah Host = [tex]2^{y} -2[/tex]

dimana y adalah jumlah biner 0 pada octet 3 dan 4 netmask

/24 bila di-binerkan menjadi:

1111 1111. 1111 1111. 1111 1111. 0000 0000

255         255        255            0

Sehingga Jumlah Host

= [tex]2^{8} - 2[/tex]

= [tex]256 - 2[/tex]

= [tex]254[/tex]

34. Tentukan kelas. subnet mask, jumlah network dan host jika dketahui IP Address network 172.168.10.0 /22 ! * Jawaban Anda​


Berdasarkan informasi IP Address network 172.168.10.0 /22, dapat dihitung informasi berikut ini:

- Kelas: Kelas B

- Subnet mask: 255.255.252.0

- Jumlah network: 4

- Jumlah host per network: 1022

Penjelasan:

- Kelas B memiliki 16 bit untuk network dan 16 bit untuk host

- Subnet mask 22 bit (diwakili oleh 255.255.252.0) menunjukkan bahwa 22 bit pertama digunakan untuk network, sehingga sisa 10 bit digunakan untuk host

- Dengan 22 bit untuk network, terdapat 2^2 = 4 network yang dapat dibentuk

- Dengan 10 bit untuk host, terdapat 2^10 - 2 = 1022 host yang dapat di-assign pada setiap network (2 host dikurangi untuk network address dan broadcast address)

36. Diketahui Alamat IP Kelas C Dengan Alamat :a. 192. 168. 200. 200 /29b. 192. 168. 10. 225. / 30TentukanSubnet Mask :IP Awal :IP Terakhir :Network :Broadcast :Kalo bisa pakai jalan.Jawab salah satu gak papa.​


a. Alamat IP 192.168.200.200/29

Subnet Mask untuk /29 adalah 255.255.255.248 karena ada 5 bit yang digunakan untuk mengidentifikasi jaringan.

Untuk menentukan IP Awal, kita tinggal menambahkan 1 pada alamat IP yang diberikan:

IP Awal = 192.168.200.201

Untuk menentukan IP Terakhir, kita harus menghitung berapa banyak host yang dapat digunakan pada subnet ini. Pada /29, terdapat 8 subnet yang dapat digunakan (2^3 = 8), tetapi 2 IP digunakan untuk Network dan Broadcast. Jadi, hanya ada 6 IP yang dapat digunakan (8 - 2). Oleh karena itu, IP Terakhir akan menjadi alamat sebelum Network pada subnet berikutnya:

IP Terakhir = 192.168.200.206

Network akan sesuai dengan alamat IP awal yang diberikan:

Network = 192.168.200.200

Broadcast akan sesuai dengan alamat IP terakhir yang diberikan:

Broadcast = 192.168.200.207
