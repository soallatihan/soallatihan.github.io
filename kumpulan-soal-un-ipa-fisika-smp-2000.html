4. Soal diutamakan untuk siswa-siswi SMP. Petunjuk: Jawablah soal menggunakan cara yang lengkap dan jelas Materi: Bilangan (Kelas 7) Jika bulan ini adalah bulan Januari, maka 5 pangkat 2000 lagi bulan apa?


Mapel : Matematika
Tingkat : Olimpiade SMA
Bab : Modulo

Pembahasan : 

Cara Pertama
[tex] 5^{1} = 5 = Juni[/tex]
[tex] 5^{2} = 25 = Februari[/tex]
[tex] 5^{3} = 125 = Juni[/tex]
[tex] 5^{4} = 625 = Februari[/tex]

Dari data diatas diambil kesimpulan setiap 2x Pola itu BERULANG.
2000 : 2 = 1000 (Habis dibagi)

Sisa = 0
[tex] 5^{0} = 1[/tex]
Maka 1 Bulan setelah Januari / 1 Bulan sebelum Pengulangan Pola adalah JAWABANNYA

Jadi... [tex] 5^{2000} lagi adalah Bulan Februari[/tex]



Cara Kedua
[tex] 5^{2000} mod 12[/tex]
[tex]( 5^{2} )^{1000}[/tex] mod 12
[tex]( 25^{1000} ) mod 12[/tex]
[tex] (12.2 + 1)^{1000} mod 12 [/tex]
[tex] 1^{1000} mod 12[/tex]
[tex]1 mod 12[/tex]

Jadi... 1 Bulan setelah Bulan Januari adalah FEBRUARI
