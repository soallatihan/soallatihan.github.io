29. Ujian mid semester kimia ktsp 2006 Larutan naoh 30,0% massa memiliki massa jenis 1,33 g/ml molaritas larutan ini adalah


[NaOH] : 30% (b/b)
massa jenis NaOH : 1,33 g/mL
M NaOH : .......... mol / L ?

Jawab

Langkah 1
Tentukan nilai % (b/v) untuk mengetahui massa gram NaOH dalam 100 mL larutan.

% (b/v) : % (b/b) x massa jenis
: 30% (b/b) x 1,33 g/mL
: 30% (g/g) x 1,33 g/mL
: 39,9% (g/mL)
: 39,9% (b/v)

39,9% (b/v) artinya 39,9 gram NaOH dalam 100 mL larutan.

Langkah 2
Tentukan jumlah Molaritas NaOH dengan menggunakan rumus sebagai berikut :

Mr NaOH : 23 + 16 + 1
: 40 g/mol

M : mol / L
M : (massa/ Mr) / V. lar
M : massa /( Mr x V.lar
: 39,9 g / (40 g/mol . 100 mL . 10^-3 L/mL)
: 9,975 mol / L
: 9,975 M

Cara cepat mengerjakan soal tersebut. Gunakan rumus berikut :

M : (% (b/b) x m. jenis x 10) / Mr
M : (30 x 1,33 x 10) / 40
M : 9,975 mol /L
