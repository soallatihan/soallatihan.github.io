1. Bilangan biner 1100000.10100001.11110000.00000111 jika dijadikan kebilangan desimal, maka akan menjadi IP Address. IP Address kelas berapa? Berikan alasan kalian. *​


Jawaban:

Penjelasan:

untuk menentukan kelas ip address, ubah biner terdepan jadi desimal :

1100000 = 0*2^0 + 0*2^1 + 0*2^2 + 0*2^3 + 0*2^4 + 1*2^5 + 1*2^6

1100000 = 32 + 64 = 96

12. berapa jumlah oktal biner dalam satu rangkaian ip address versi 4​


Jawaban:

Dalam satu rangkaian alamat IP versi 4 (IPv4), terdapat empat bagian oktal yang disebut oktet. Setiap oktet direpresentasikan dalam bentuk bilangan biner dengan panjang 8 bit. Jadi, jumlah total bit dalam alamat IP versi 4 adalah 4 oktet * 8 bit/oktet = 32 bit.

Sebagai contoh, alamat IP versi 4 "192.168.0.1" dapat diuraikan menjadi biner sebagai berikut:

- 192: 11000000

- 168: 10101000

- 0: 00000000

- 1: 00000001

Total bitnya adalah 32 (8 bit + 8 bit + 8 bit + 8 bit).

15. tuliskan bentuk biner dari kelas ip address 220.221.100.60! tuliskan langkah " pengerjain nya​


Jawaban:

Untuk menulis bentuk biner dari kelas IP address 220.221.100.60, kita perlu mengikuti langkah-langkah berikut:

- Pertama, kita perlu mengetahui bahwa kelas IP address 220.221.100.60 adalah kelas **C**, karena angka pertama (220) berada dalam rentang 192-223.

- Kedua, kita perlu mengetahui bahwa kelas C memiliki **24 bit** untuk bagian jaringan dan **8 bit** untuk bagian host. Ini berarti bahwa tiga oktet pertama (220.221.100) merepresentasikan jaringan dan oktet terakhir (60) merepresentasikan host.

- Ketiga, kita perlu mengubah setiap oktet menjadi bentuk biner dengan menggunakan metode pembagian berulang. Misalnya, untuk mengubah 220 menjadi biner, kita bisa melakukan hal berikut:

- Bagi 220 dengan 2 dan catat sisa pembagiannya (0).

- Bagi hasil pembagian sebelumnya (110) dengan 2 dan catat sisa pembagiannya (0).

- Ulangi langkah ini sampai hasil pembagian menjadi nol.

- Susun sisa pembagian dari bawah ke atas untuk mendapatkan bentuk biner.

Dengan cara ini, kita bisa mendapatkan bentuk biner dari 220 sebagai berikut:

| Pembagian | Sisa |

| --- | --- |

| 220 / 2 = 110 | 0 |

| 110 / 2 = 55 | 0 |

| 55 / 2 = 27 | 1 |

| 27 / 2 = 13 | 1 |

| 13 / 2 = 6 | 1 |

| 6 / 2 = 3 | 0 |

| 3 / 2 = 1 | 1 |

| 1 / 2 = 0 | 1 |

Bentuk biner dari 220 adalah **11011100**.

- Keempat, kita perlu mengulangi langkah ketiga untuk setiap oktet yang tersisa dan menambahkan titik antara setiap oktet biner. Dengan cara ini, kita bisa mendapatkan bentuk biner dari kelas IP address sebagai berikut:

- Bentuk biner dari 221 adalah **11011101**.

- Bentuk biner dari 100 adalah **01100100**.

- Bentuk biner dari 60 adalah **00111100**.

- Bentuk biner dari kelas IP address adalah **11011100.11011101.01100100.00111100**.

Jadi, bentuk biner dari kelas IP address 220.221.100.60 adalah **11011100.11011101.01100100.00111100**.

24. mengapa IP Address dikatakan terdiri dari 32 bit biner​#plis jawab kak


Jawaban:

IP address adalah sebuah sistem pengalamatan unik setiap host yang terkoneksi ke jaringan berbasis TCP/IP. IP address bisa dianalogikan seperti sebuah alamat rumah. Ketika sebuah datagram dikirim, informasi alamat inilah yang menjadi acuan datagram agar bisa sampai ke device yang dituju. IP Address terbagi dalam 2 versi, IPv4 dan IPv6. Sebuah IP address versi 4 atau IPv4 terbentuk dari 32 binary bits. Dari 32 binary bits tersebut terbagi lagi menjadi 4 octet (1 octet = 8 bits). Nilai tiap oktet diatara 0 sampai 255 dalam format desimal, atau 00000000 - 11111111 dalam formal binary. Setiap octet dikonversi menjadi desimal dan dipisahkan oleh tanda titik (dot). Sehingga format akhir IP address biasanya berupa angka desimal yang dipisahkan dengan tanda titik, contohnya 172.16.254.1.

34. Ip address versi 4 (ipv4) merupakan kumpulan bilangan biner yang terdiri dari…


Jawaban & Penjelasan:

Alamat IP terdiri dari 32 bit.

Alamat IP terdiri dari 32 bit, sering ditampilkan sebagai 4 oktet angka dari 0-255 yang direpresentasikan dalam bentuk desimal, bukan dalam bentuk biner. Misalnya, alamat IP: 168.212.226.204 dalam bentuk biner adalah 10101000.11010100.11100010.11001100.

Internet Protocol versi 4 (IPv4) mendefinisikan alamat IP sebagai angka 32-bit. Namun, karena pertumbuhan Internet dan menipisnya alamat IPv4 yang tersedia, versi baru IP (IPv6), menggunakan 128 bit untuk alamat IP, dikembangkan pada tahun 1995, dan distandarisasi pada bulan Desember 1998.

Ukuran awalan, dalam bit, ditulis setelah miring. Ini disebut "notasi garis miring". Ada total 32 bit dalam ruang alamat IPv4. Misalnya, jika sebuah jaringan memiliki alamat "192.0.2.0/24", angka "24" mengacu pada berapa banyak bit yang terkandung dalam jaringan. Ini berarti alamat host tunggal. Itu disebut notasi CIDR. 192.168.0.1 mewakili IP dan /32 mewakili jumlah bit dalam mask.
